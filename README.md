# Supply Chain & Delivery Performance Analysis

## Overview

This project analyzes supply chain operations to identify inefficiencies in delivery performance, inventory management, and logistics cost.
The analysis was carried out using **Python for data preprocessing, SQL for business insights, and Power BI for visualization**.

---

## Tools & Technologies

* **Python** – Data cleaning and preprocessing
* **MySQL** – Data analysis using SQL queries
* **Power BI** – Interactive dashboard and visualization

## Key Insights

* Certain regions contribute significantly higher sales compared to others
* Transportation costs vary across suppliers, indicating optimization opportunities
* Delivery delays are observed in specific regions
* Backorders highlight gaps in inventory planning
* Warehouse utilization is uneven across locations

---

## Business Recommendations

* Improve delivery performance by optimizing logistics in high lead-time regions
* Reduce transportation costs by reviewing supplier and routing strategies
* Minimize backorders through better demand forecasting and inventory planning
* Balance warehouse utilization by redistributing inventory
* Focus on high-performing regions and categories for better revenue growth
